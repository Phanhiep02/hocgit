# Terms

repository (repo)
branch(nhánh)

# Commands

- git init
- git status (trạng thái )
- git add (lưu lại folder)
- git reset (quay lại folder vừa lưu)
- git commit -m (note)
- git log --oneline
- git checkout (id commit) để tới nhánh commit đó
- git branch - nhánh mặc định (main)
- git checkout -b tên của branch để tạo nhánh mới
